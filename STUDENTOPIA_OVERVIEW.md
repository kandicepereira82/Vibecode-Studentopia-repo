# Studentopia - Complete App Overview

## üì± App Purpose & Vision

**Studentopia** is a comprehensive mobile productivity companion designed to help students (ages 12-25) stay organized, maintain focus, manage stress, and achieve academic success through a delightful, gamified experience.

### Core Mission
Transform studying from a stressful chore into an engaging, supported journey with a personalized AI-powered companion by your side.

### Target Audience
- **Primary:** Students aged 12-25 (middle school, high school, college)
- **Secondary:** Teachers managing student groups and tracking progress
- **Parents:** Can monitor multiple children's calendars and study schedules

## üë§ User Journey

### First-Time User Flow

**Step 1: Onboarding (30-60 seconds)**
1. See app icon and "Studentopia" title
2. Enter name and email (optional)
3. Choose animal companion from 25 options (5√ó5 grid)
4. Select theme color from 8 options with live preview
5. Choose role: Student or Teacher
6. See celebration animation

**Step 2: Authentication (30 seconds)**
1. Choose between Login or Signup tabs
2. Signup: Enter email, password, confirm password
3. See success message and autofill confirmation
4. Automatically logged in

**Step 3: Home Screen Welcome (First impression)**
1. See personalized greeting with their name
2. Companion appears with welcome message
3. Empty state encouraging adding first task
4. Quick tour of main features
5. Week calendar showing current week

**Step 4: Add First Task (30 seconds)**
1. Tap "Add Task" from home or navigate to Tasks tab
2. Fill in task details (title, description, due date, category)
3. Optional: Set reminder time
4. Save task
5. See task appear on home screen and calendar
6. Companion celebrates first task

**Step 5: Start First Study Session (2 minutes)**
1. Navigate to Timer tab
2. See default 25-minute Focus timer
3. Optional: Customize duration or select alarm sound
4. Press Start button
5. See countdown begin with theme colors
6. Companion encourages focus
7. Complete session and see stats update

**Step 6: Explore Features (Self-directed)**
- Check Calendar sync options
- Explore AI Helper with sample question
- Read a Study Tip
- Customize companion appearance
- Try Breathwork exercise
- Browse Groups/Live Sessions (if interested)

### Returning User Flow

**Daily Routine:**

1. **Morning:**
   - Open app, see daily study reminder with companion's greeting
   - Review today's tasks on home screen week calendar
   - Check task due dates and priorities
   - Plan study sessions for the day

2. **Study Session:**
   - Start Focus timer (25 min)
   - Work on highest priority task
   - Check off completed task
   - See companion celebrate
   - Take 5-min break with breathwork
   - Repeat cycle 2-4 times

3. **During Study:**
   - Use AI Helper for stuck homework problems
   - Quick reference Study Tips for better strategies
   - Background music playing from playlist
   - Task reminders notify of upcoming deadlines

4. **Evening:**
   - Review completed tasks and streak
   - Add tomorrow's tasks to calendar
   - Check group updates if in study groups
   - Optional: Join Live Study Session with friends

5. **Weekly:**
   - Review week calendar for upcoming deadlines
   - Sync tasks to Google/Apple Calendar
   - Check study minutes and streak stats
   - Adjust daily reminder time if needed

### Teacher User Flow

**Setup:**
1. Complete onboarding and select "Teacher" role
2. Navigate to Groups tab
3. Create first group/class:
   - Enter school name, class name, teacher email
   - Read and accept group rules
   - Receive 6-character share code
4. Share code with students via QR code or text

**Daily Usage:**
1. View all created classes in Groups tab
2. Tap class to see members list
3. View group analytics:
   - Total members
   - Active members (7 days)
   - Average completion rate
   - Top performers leaderboard
   - Students needing support
4. Monitor individual student progress
5. Optional: Assign tasks to group (future feature)

**Live Sessions (Optional):**
1. Switch to "Live Sessions" tab within Groups
2. Create study room for class
3. Invite students
4. Control synchronized timer
5. Monitor chat and participants
